# EDA-with-SQL-Notebook
---

## Exploratory Data Analysis (EDA) on Chicago Datasets

This project performs **exploratory data analysis (EDA)** on multiple datasets from **Chicago**, including **socioeconomic indicators, public schools, and crime data**, to extract meaningful insights and solve real-world problems. The analysis combines **SQL querying via SQLite in JupyterLite** with **Python tools** to manipulate, summarize, and visualize the data efficiently.

### **Key Features**

* **Data Integration:** Combined multiple Chicago datasets to enable cross-dataset analysis.
* **Data Cleaning & Preparation:** Handled missing values, inconsistent entries, and normalized datasets for accurate analysis.
* **Exploratory Analysis with SQL & Python:**

  * **SQLite in JupyterLite:** Used for filtering, aggregations, joins, and subqueries across datasets.
  * **Python (Pandas & PrettyTable):** Processed query results, generated structured tables, and summarized insights.
* **Data Visualization:** Generated charts and tables to understand distributions, correlations, and patterns (optional integration with pandas plotting if needed).
* **Insights & Problem Solving:**

  * Analyzed **crime trends** across neighborhoods and identified high-risk areas.
  * Explored **socioeconomic indicators** to understand disparities and community patterns.
  * Investigated **public school performance** in relation to demographics and neighborhood data.
  * Answered practical questions such as:

    * Which neighborhoods have the highest crime rates?
    * How do socioeconomic factors correlate with crime or school performance?
    * What trends can be observed across different datasets when combined?

### **Tech Stack**

* **SQL (SQLite in JupyterLite):** Querying, joins, and aggregations for structured data analysis.
* **Python (Pandas, PrettyTable):** Data wrangling, tabular presentation, and summaries.
* **Datasets:** Chicago Open Data Portal – socioeconomic indicators, crime data, and public school datasets.

### **Outcome**

This project demonstrates **end-to-end EDA using SQL and Python**, turning raw datasets into actionable insights. By combining **programmatic analysis with structured querying**, it highlights trends, relationships, and patterns across multiple real-world datasets, providing a solid foundation for decision-making and further predictive modeling.
